如何在 Plesk log 顯示真實 IP 而不是 Cloudflare 的 IP

發表於2023-01-31|更新於2023-01-31
|閱讀量:

在幫客戶看網站負載的時候，看 Plesk Log，由於有使用 Cloudflare 的 CDN 服務，會看到的都是 Cloudflare 的 IP 而不是訪客的真實 IP，所以紀錄一下如何設定，讓 Plesk log 顯示真實 IP 而不是 Cloudflare 的 IP 。

步驟：依序點選下面部分

  1. Websites & Domains
  2. Hosting & DNS
  3. Apache & nginx Settings
  4. 最下面的 Apache & nginx Settings 輸入以下內容

如何知道 Cloudflare IP Ranges
