macOS 12 Monterey 重新安裝記錄

因為自己的 MacBook Pro 2017（A1708）空間不夠，前年自己換了 SSD 後，每次開機都要等 5 - 10 分鐘，而且三不五時從休眠狀態回來會重開機，但是由於專案一直很忙，沒空重灌，也擔心是換的SSD的問題，就一直懶得處理，直到前幾天看到 OWC 有出相容 SSD（Aura Pro NT），想說就重灌試試看，如果重灌後狀況還是沒有改變的話，應該就是硬體相容性問題，就可以考慮看看要不要再買 OWC 出的 SSD，或是可以開始存錢買新的….

記錄一下，乾淨的重安裝，不是從 Time Machine 還原的過程。

如何製作可開機的 macOS 安裝磁碟

就看官方文件沒有什麼好說的：
https://support.apple.com/zh-tw/HT201372

安裝完後發現開機問題就解決啦！開心！可以再戰好多年

記錄安裝的程式

由於要設定平常使用的工作環境所以記錄一下安裝的東西

  • 工作需要

    • Edge：去年看了現實的風向已經在變了？ 為什麼 Microsoft Edge 的聲勢在國外開始水漲船高？後就從 Chrome 跳了，使用到目前還滿滿意的。

    • iTerm2：終端機，新的系統預設是 zsh 了

      • Zim：據說比較快
        • theme：powerlevel10k

    • Homebrew：努力儘量各種都用他來裝

      • git：如何以 Homebrew 取代 MacOS 內建 Git 並更新
      • nvm：忘記看那邊了，網路上 google 都可以找到
      • yarn：忘記看那邊了，網路上 google 都可以找到

    • ImageOptim：圖片減肥，要上傳到網站的圖片都會用他先壓縮過

    • Paw：測試 API 用的，好用，但是我少用

    • PhpStorm：就…一整天幾乎都會用到他。

      • 就改改顏色、字體大小，其他需要才裝

    • SnippetsLab：一些常用的 Code 可以彙整起來，很久之前就有看到，但是沒有買，去年黑五特價訂閱到一年便宜的Setapp，裡面有就拿來用了，有跟 Alfred 整合，可以直接搜尋，可以同步到 gist，應該會滿方便的，

    • VScode（延伸模組如下）

      • Auto Close Tag
      • AutoFileName
      • Beautify
      • Git Graph
      • GitLens
      • indent-rainbow
      • Live Server
      • Markdown Preview
      • markdownlint
      • Settings Sync

  • 通訊軟體

    • LINE：各種用途
    • Slack：公司用
    • Teams：跟客戶用
    • Discord：

  • 專案管理

    • Trello：公司用的事項管理，可參考（https://trello.com/b/dOrYQNPr/trello）
    • Timemator：時間記錄的，今年打算來記錄一下自己的工作狀況

  • 其他

    • AdGuard：擋廣告的，沒有蓋版廣告的網路真的很爽！！就算不用付費的，也建議瀏覽器裝個擋廣告的外掛，或是使用 Brave 瀏覽器，我看我同事用好像很不錯。
    • Authy：兩步驟驗證，資料都同步了，開啟輸入密碼就回來了，不像 Google Authenticator 換手機要全部重來，不過 Google Authenticator 好像也不能裝在 MacBook 上
    • Alfred：一些平常小操作都能透過他完成，例如切換藍芽連線裝置，切換聲音播放裝置，儘量讓自己的手不離開鍵盤處理各種事情，如果有好用的 Workflow 還請推薦。
    • BetterTouchTool：也是一些小功能，主要拿來當作視窗大小的調整，靠左靠右最大指定比例等，大概就是拿來當Magnet、Mosaic 同樣的功能，但是他還能設定很多，例如快速鍵在資料夾建立新檔案，一樣是要減少手離開鍵盤。
    • Camtasia 3：超強螢幕錄製，跟客戶做後臺教學的時候會做螢幕錄影，之後給連結，忘記了可以先看，幾年前特價的時候買到，雖然已經不是最新版本，不能錄製完直接分享到 Youtube，不過還是很好用，參考介紹：沒人敢稱第一的螢幕錄製/剪輯軟體| Camtasia Studio 教學
    • MoneyWiz3：記帳用
    • Setapp：訂閱到的，裡面有很多好用的 App，看到有趣的就會轉過來玩玩看，目前有裝 CleanMyMac X，還滿不錯的。
    • Sticky Password：密碼管理的，想說看能不能統一管理密碼，不過目前使用上沒有感覺特別的便利，手機上好像不能靠他自動填入密碼，一些 APP 太久沒有要重新登入，還是常常按忘記密碼。
    • MicroSoft To Do：個人使用的 todolist，只有越來越長而已，總是消的不夠快。
    • 鼠鬚管：一樣是透過 Homebrew 安裝
      • 洋蔥版的地球拼音： https://github.com/oniondelta/Onion_Rime_Files ，由於我是用拼音輸入，所以另外找了洋蔥的地球拼音來用，真的大推！！
      • 網址列內按方向鍵或倒退鍵會錯誤清空拼字，相見恨晚——鼠鬚管洋蔥版純注音輸入法，主要就是在 squirrel.custom.yaml 改 style/inline_preedit: true
      • 自己把韓文拿掉：因為我不會韓文啊，講的好像我會日文一樣…，修改 terra_pinyin_onion.extended.dict.yaml，把有kr的註解掉，重新部署
      • 同步設定的方式可以看這篇 https://github.com/Ponpon55837/Squirrel

大概就這樣吧，有趣的事情是，重灌前我的空間大約只剩下 256G，重灌後，程式都裝的出差不多了，空間卻還有 420G ，之前的 170G 究竟是什麼……
